📊 Calculadora de IMC

Aplicação simples que calcula o IMC (Índice de Massa Corporal) a partir da altura e peso informados pelo usuário, retornando o valor e sua classificação (abaixo do peso, normal, sobrepeso ou obesidade).

🚀 Funcionalidades 

Entrada de peso e altura
Cálculo automático do IMC
Exibição do resultado e classificação
🧮 Fórmula

IMC = peso / (altura²)

🛠️ Tecnologias

(python)
